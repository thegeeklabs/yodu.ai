# Yodu.ai

A generic purpose Recommender System that can be configured for any UseCase.
ExampleUseCase:

- Social Platforms
- Ecommerce Websites
- Video Portals
- News Aggregator Websites

If you want to contribute just email me at shashank[at]yodu.ai . Happy to share DB and other access.
